# Zhiyuan (Jean) Yao — Academic Website

A lightweight static academic portfolio focused on geospatial data science, transportation, accessibility, and rural infrastructure.

## Preview

Open `index.html` in a browser. No build step is required.

## Publish with GitHub Pages

1. Create a public GitHub repository.
2. Push the contents of this folder to its default branch.
3. In the repository, open **Settings → Pages**.
4. Choose **Deploy from a branch**, select the default branch and the root folder, then save.

The site is responsive and uses only HTML, CSS, and a small amount of JavaScript.
